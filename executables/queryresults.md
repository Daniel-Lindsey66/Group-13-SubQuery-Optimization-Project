# Query Analysis

## Crime Dataset

- **Query 1:** Retrieves records where the victim's age is greater than the average victim age of the entire dataset.

- **Query 2:** Locates crime records in areas where a "Theft of Identity" crime has already occurred.

- **Query 3:** Returns areas where more than 100 crimes have occurred.

- **Query 4:** Returns crime records which are the most recent in their area.

- **Query 5:** Returns crime records in areas where "UNAUTHORIZED COMPUTER ACCESS' crime hasn't occured.

## Complaint Dataset

- **Query 1:** Returns the delay between create_date/issue_date along with the average delay of create_date/issue_date for complaints in the same city.

- **Query 2:** Returns cities where the total number of complaints exceeds the number of complaints that were submitted via the Internet.

- **Query 3:** Returns each complaint record along with the creation date of the most recent complaint in the same zip code.

- **Query 4:** Returns a complaint record along with the a count of complaints in the same zip code.

- **Query 5:** Finds complaints that share a caller_id with at least 1 other complaint.
